1. BMTC Bus Ticket Verification System

A full-stack web application that simulates a BMTC bus ticket verification system using OTP-based validation.

This project was built as a learning-focused application to understand full-stack development, backend APIs, and basic authentication flows.



2. Features
- OTP generation and verification
- Aadhaar-based passenger verification (simulated)
- Backend API using Node.js & Express
- Frontend interface for user interaction
- Secure request-response flow


3. Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Tools:** Git, GitHub, Postman
